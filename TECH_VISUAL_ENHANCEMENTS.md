# 科技感视觉优化

## 🎨 优化概述

将原本纯黑色的背景升级为富有层次感和动态效果的赛博朋克风格，打造沉浸式的高科技 IDC 监控中心体验。

## 🌌 背景优化

### 1️⃣ **动态渐变背景**

#### 优化前
- ❌ 纯黑色背景 `#0A0A0B`
- ❌ 单调无层次
- ❌ 静态无变化

#### 优化后
- ✅ **多层渐变**：5 色动态渐变
  ```css
  #0a0e1a (深蓝黑) → 0%
  #0f1428 (深蓝) → 25%
  #0a0f20 (深青黑) → 50%
  #0f1428 (深蓝) → 75%
  #0a0e1a (深蓝黑) → 100%
  ```
- ✅ **动态流动**：30 秒周期渐变动画
- ✅ **400% 背景尺寸**：平滑过渡效果

#### 视觉效果

## 🎬 动画效果

### 1. **渐变流动** (Gradient Shift)
```css
@keyframes gradient-shift {
  0%   { background-position: 0% 50%; }
  50%  { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}
```
- **时长**：20 秒
- **缓动**：ease
- **效果**：背景色缓慢流动

### 2. **光晕浮动** (Orb Float)
```css
@keyframes orb-float {
  0%, 100% { transform: translate(0, 0) scale(1); }
  25%      { transform: translate(50px, -50px) scale(1.1); }
  50%      { transform: translate(-50px, 50px) scale(0.9); }
  75%      { transform: translate(50px, 50px) scale(1.05); }
}
```
- **时长**：20 秒
- **效果**：三个光晕球独立浮动

### 3. **扫描线移动** (Scanline Move)
```css
@keyframes scanline-move {
  0%   { transform: translateY(0); }
  100% { transform: translateY(10px); }
}
```
- **时长**：10 秒
- **效果**：细微的扫描线垂直移动

### 4. **网格脉冲** (Grid Pulse)
```css
@keyframes grid-pulse {
  0%, 100% { opacity: 0.2; }
  50%      { opacity: 0.5; }
}
```
- **时长**：4 秒
- **效果**：网格线明暗呼吸

## 🚀 性能优化

### 1. **GPU 加速**
- 所有动画使用 `transform` 和 `opacity`
- 避免使用 `top`, `left`, `width`, `height`

### 2. **分层渲染**
- 背景层使用 `will-change: transform`
- 内容层独立，避免重绘

### 3. **动画优化**
- 背景渐变：30 秒周期（低频）
- 光晕浮动：20 秒周期（低频）
- 扫描线：10 秒周期（中频）
- 网格脉冲：4 秒周期（高频但简单）
